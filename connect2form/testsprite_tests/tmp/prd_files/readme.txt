== Installation ==

1. Upload the plugin files to `/wp-content/plugins/connect2form/` directory, or install through WordPress admin
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Contact Forms > Add New to create your first form
4. Use the shortcode `[connect2form id="1"]` to display your form

**Minimum Requirements:**
* WordPress 5.0 or higher
* PHP 7.4 or higher
* MySQL 5.6 or higher

== Developer Information ==

**Hooks & Filters:**

connect2form provides extensive customization through WordPress hooks:

* `connect2form_form_data_before_save` - Filter form data before saving
* `connect2form_submission_data` - Filter submission data
* `connect2form_email_template` - Customize email templates
* `connect2form_validation_rules` - Add custom validation
* `connect2form_integration_settings` - Modify integration settings

== Privacy Policy ==

connect2form takes privacy seriously:

* We don't collect any personal data unless explicitly configured
* Form submissions are stored in your WordPress database
* Integration data is sent only to configured services
* No data is sent to external servers without explicit configuration
* You maintain full control over all data
